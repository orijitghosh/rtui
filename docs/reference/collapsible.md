# Create a collapsible section

Create a collapsible section

## Usage

``` r
collapsible(title, ..., collapsed = TRUE, id = NULL, classes = NULL)
```

## Arguments

- title:

  Section title.

- ...:

  Child widget specs.

- collapsed:

  Initial collapsed state.

- id:

  Optional widget id.

- classes:

  Optional CSS classes (character vector).

## Value

An `rtui_spec` list.
