# Create a checkbox widget

Create a checkbox widget

## Usage

``` r
checkbox(label, value = FALSE, id = NULL, classes = NULL, tooltip = NULL)
```

## Arguments

- label:

  Checkbox label text.

- value:

  Initial checked state (logical).

- id:

  Optional widget id.

- classes:

  Optional CSS classes (character vector).

- tooltip:

  Optional tooltip text shown on hover.

## Value

An `rtui_spec` list.
