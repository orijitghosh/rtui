# Create an append-only log view widget

Create an append-only log view widget

## Usage

``` r
log_view(id = NULL, classes = NULL, max_lines = 1000L)
```

## Arguments

- id:

  Optional widget id.

- classes:

  Optional CSS classes (character vector).

- max_lines:

  Maximum number of lines to retain.

## Value

An `rtui_spec` list.
