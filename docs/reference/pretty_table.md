# Create a pretty table widget (rich-formatted)

Create a pretty table widget (rich-formatted)

## Usage

``` r
pretty_table(df, title = NULL, id = NULL, classes = NULL)
```

## Arguments

- df:

  A data.frame to display.

- title:

  Optional table title.

- id:

  Optional widget id.

- classes:

  Optional CSS classes (character vector).

## Value

An `rtui_spec` list.
