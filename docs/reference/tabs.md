# Create a tabbed pane container

Create a tabbed pane container

## Usage

``` r
tabs(..., id = NULL, classes = NULL)
```

## Arguments

- ...:

  Tab pane specs created with
  [`tab_pane()`](https://orijitghosh.github.io/rtui/reference/tab_pane.md).

- id:

  Optional widget id.

- classes:

  Optional CSS classes (character vector).

## Value

An `rtui_spec` list.
