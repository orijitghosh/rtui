# Create a grid layout

Create a grid layout

## Usage

``` r
grid(..., rows = NULL, cols = NULL, id = NULL, classes = NULL)
```

## Arguments

- ...:

  Child widget specs.

- rows:

  Number of rows (integer or NULL).

- cols:

  Number of columns (integer or NULL).

- id:

  Optional widget id.

- classes:

  Optional CSS classes (character vector).

## Value

An `rtui_spec` list.
