# Create a multi-line text area widget

Create a multi-line text area widget

## Usage

``` r
text_area(
  value = "",
  language = NULL,
  show_line_numbers = FALSE,
  id = NULL,
  classes = NULL
)
```

## Arguments

- value:

  Initial text content.

- language:

  Optional language for syntax highlighting.

- show_line_numbers:

  Show line numbers.

- id:

  Optional widget id.

- classes:

  Optional CSS classes (character vector).

## Value

An `rtui_spec` list.
