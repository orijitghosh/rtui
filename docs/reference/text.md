# Create a text widget

Create a text widget

## Usage

``` r
text(content, id = NULL, classes = NULL, tooltip = NULL)
```

## Arguments

- content:

  Character string to display.

- id:

  Optional widget id.

- classes:

  Optional CSS classes (character vector).

- tooltip:

  Optional tooltip text shown on hover.

## Value

An `rtui_spec` list.
